# 食用须知

这是博文 webpack 混淆知识点案例代码，配合博文一起食用风味更佳

<br/>

# 如何运行

### 1.安装 node_modules

```bash
npm install
```

或

```bash
yarn
```

### 2.打包生产环境代码

```bash
npm run build
```

<br/>

# 目录结构

1.  案例代码均在 /src 目录下，可以自行修改运行
2.  打包生成的文件均在 /dist 目录下
3.  配置文件均在 /configs 目录下，大部分的配置都在 webpack.common.js 中，sourses-map 配置在 webpack.prod.js 中
